# 02_詳細設計

`/sf-design` → `[1] 詳細設計` が生成するドキュメント。

| ファイル | 生成元 | 用途 |
|---|---|---|
| `{機能グループ名}_詳細設計.xlsx` | `generate_detail_design.py` | 業務機能グループ単位の詳細設計（6シート） |

**6シート構成**:
1. 概要 — FG基本情報・スコープ・トリガー
2. 業務フロー — 業務の流れ（スイムレーン）
3. 対象オブジェクト — 使用オブジェクト・項目マッピング
4. 処理概要 — ロジック・条件分岐・計算式
5. 関連コンポーネント — Apex/Flow/LWC の紐づき
6. 影響範囲 — 関連機能・依存関係・テスト観点

**前提**: `/sf-memory` でカテゴリ5（機能グループ定義）を実行して `docs/_src/feature_groups.yml` が存在すること。
