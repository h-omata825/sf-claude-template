assistantエージェントとして、テンプレートのアップグレードを実行してください。

## 概要

`scripts/upgrade.sh` を実行してテンプレートを更新する。
スクリプトが差分検出・確認・適用・クリーンアップを全て行う。

## ユーザー入力

$ARGUMENTS

- **引数なし** → デフォルト（developブランチ）
- **タグ/ブランチ指定** → そのバージョンを取得

## 実行

```bash
bash scripts/upgrade.sh $ARGUMENTS
```

スクリプトが対話的に確認を行うので、そのまま実行させる。

## 実行後

1. スクリプトの出力結果をユーザーに伝える
2. 変更があった場合は `git diff .claude/` で変更内容を確認するよう提案する
