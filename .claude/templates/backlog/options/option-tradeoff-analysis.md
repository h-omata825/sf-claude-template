# option-tradeoff-analysis

## 何をするか

各対応案のメリット・デメリットをコスト・工期・将来影響まで含めて深掘りする。option-alternative-approaches で複数案が出た場合に続けて実行する。

## 実行手順

1. 対象となる全ての案（A / B / C）を approach-plan.md から確認する
2. 各案について以下の観点を深掘りする:

   **短期コスト**:
   - 実装工数（見積時間）
   - テスト工数
   - レビュー・確認工数

   **長期影響**:
   - メンテナンス容易性（将来の修正がしやすいか）
   - 類似要望が来た時の再利用性
   - 技術的負債の増減

   **リスク**:
   - 既存機能への影響範囲
   - ロールバック容易性
   - 本番リリース後の問題発生確率

3. 比較結果を踏まえて最終推奨案の根拠を明文化する
4. approach-plan.md の「推奨案と根拠」セクションを更新する
