# 権限セット / プロファイル（Permission Sets / Profiles）

## 権限セット作成時

1. 既存の権限セット構成を確認（類似の権限セットが既にないか）
2. **プロファイルは原則変更しない**: 新しい権限は権限セットで追加するのがベストプラクティス

**確認事項:**
- 対象ユーザー: どのロール・部署のユーザーに付与するか
- オブジェクト権限（CRUD）: 必要最小限の権限のみ付与
- FLS: 参照のみ or 編集も可能か
- システム権限: 「全データの編集」等の強い権限は特に注意、用途を確認
- 権限セットグループへの追加: 既存グループに組み込むか

**変更時の注意:** 権限の削除は既存ユーザーの業務影響がないか確認してからユーザーに判断を委ねる
