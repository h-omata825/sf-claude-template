# メールテンプレート（Email Templates）

## 作成時

1. `docs/data/email-templates.md` に既存テンプレートがあればトーン・文体・差し込み項目の使い方を参照
2. `docs/overview/org-profile.md` の会社名・担当者署名の慣例を確認

**確認事項:**
- テンプレート種別: テキスト / HTML / Visualforce（リッチなレイアウトが必要か）
- 差し込み項目: 使用する差し込み項目を一覧提示し、nullの場合の代替文字列（`{!Contact.Name}` が空の場合の `「ご担当者様」` 等）を提案
- 件名: 差し込み項目を使うか確認
- フォルダ: どのフォルダに保存するか確認
- 言語: 多言語対応が必要か
